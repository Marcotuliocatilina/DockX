prova primo commit

* per committare  :  git commit -m "Primo commit"
* test file modificato on line

* git pull                  per scaricare le modifiche da server
* git restore readme.txt    per ripristinare la versione del file prima delle modifiche locali
* git status                indica quali file sono stati modificati
* git add .                 per aggiungere tutti i file modificati
* git commit -m "Descrizione chiara delle modifiche"            per commentare il commit
* git push origin main      per allineare il repository remoto
